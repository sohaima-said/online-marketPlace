Online Marketplace & Retail Management System
A desktop E-Commerce application built using Java Swing/AWT and core Object-Oriented Programming (OOP) principles to simulate an interactive digital store environment.
Key Features
Multi-Role Authentication: Separate secure login dashboards for both Admins and Customers.
Inventory & Stock Control: Dynamic stock handling and classification for diverse product categories (Clothing, Electronics, Groceries).
Cart & Checkout System: Fully tracks user shopping carts, manages item additions, and dynamically calculates totals.
Flexible Payments: Modular architecture simulating both Cash and Credit Card checkout verification pipelines.
Data Persistence: Saves and loads application states locally using background binary data storage files (.dat).
Tech Stack & Concepts
Language: Java
GUI Framework: Java Swing / AWT
Core Concepts: Object-Oriented Programming (Inheritance, Polymorphism, Encapsulation), File Handling, Event-Driven UI Logic.
How to Setup & Edit
Clone the project:
Bash
